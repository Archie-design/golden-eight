# 夥伴系統使用說明

定課夥伴系統讓你與最多 5 位夥伴互相督促、追蹤彼此的打卡進度，並在每日打卡完成時送上一句鼓勵。完成不同條件還可解鎖 9 項專屬成就。

---

## 主要功能

| 功能 | 說明 |
|------|------|
| **雙向邀請** | 邀請對方並等待接受；對方未接受前不會建立夥伴關係 |
| **打卡狀態同步** | 看見每位夥伴今日是否已打卡、完成哪 8 項任務、本月達成率、目前破曉連續天數 |
| **每日鼓勵** | 每天可送出一句預設訊息給每位夥伴；對方會在卡片上看到你傳的內容 |
| **夥伴成就** | 4 大類共 9 個專屬成就（社交、競爭、同步、鼓勵） |
| **Dashboard 動態區** | 儀表板上方精簡顯示前 3 位夥伴狀態，一眼掌握 |

---

## 系統入口

導覽列「**夥伴**」分頁，連結到 `/partners` 頁面，含三個分頁：

1. **我的夥伴** — 已建立的夥伴清單與每日動態
2. **邀請管理** — 收到的邀請（接受/拒絕）+ 我送出的邀請（取消）
3. **尋找夥伴** — 搜尋名字、送出邀請

Dashboard 也會顯示「夥伴動態」區塊，最多顯示 3 位，點「查看全部」回到完整頁面。

---

## 操作流程

### 一、找到並邀請夥伴

1. 進入 **夥伴 → 尋找夥伴** 分頁
2. 輸入對方姓名（可模糊搜尋）
3. 找到目標後按「**邀請**」
4. 對方會在「**邀請管理 → 收到的邀請**」看到你的邀請

**規則**：
- 不能邀請自己
- 對方必須是活躍成員
- 不能對同一人重複邀請（已有 pending 或 accepted 都會被擋）
- 對方若曾拒絕過你，需等待 **7 天冷卻**後才能再邀請
- 雙方任一方達到 **5 人上限**時無法建立新關係

### 二、收到邀請

進入 **邀請管理 → 收到的邀請**：
- 按「**接受**」→ 建立夥伴關係，雙方清單同步出現對方
- 按「**拒絕**」→ 對方 7 天內無法重邀

接受邀請會立刻觸發雙方的社交類成就計算（PARTNER_FIRST / 3 / 5）。

### 三、查看夥伴與每日鼓勵

在 **我的夥伴** 看到每位夥伴的卡片：
- **姓名 + 等級**（黃金/白銀/青銅）
- **今日打卡狀態**（綠色「已打卡」/ 灰色「未打卡」）
- **8 項任務小圈**（綠色 = 完成、灰色 = 未完成；今日未打卡時為空灰）
- **本月達成率**（含進度條）
- **破曉連續天數**（有 streak 時顯示火焰圖示）

**送出鼓勵**：
1. 點卡片底部的「**送出鼓勵**」（粉色按鈕）
2. 浮出 8 條預設訊息，選一條：「加油！」「你做得到 💪」「今天也辛苦了 🌸」「一起前進 🏃」「為你打氣 📣」「太棒了 ✨」「明天繼續 🌅」「我也在堅持 🤝」
3. 送出後按鈕變綠色「**今日已鼓勵**」（每對成員每日限 1 次）

**收到鼓勵**：對方送出後，你的夥伴卡上會出現粉色 banner「**收到鼓勵：**...」顯示對方選的訊息。

### 四、取消邀請 / 解除關係

- **取消尚未回應的邀請**：邀請管理 → 我送出的邀請 → 「取消」
- **解除已建立的夥伴關係**：我的夥伴 → 卡片右上紅色 ✕ 按鈕 → 確認對話框 → 確定

解除後：
- 雙方清單同步移除對方
- 已送出的鼓勵紀錄**保留**（屬於歷史資料）
- 已解鎖的成就**保留**（視為歷史里程碑）
- 同步打卡天數歸零（若重新成為夥伴會從零起算）

## 9 個夥伴成就

| 類別 | Code | 名稱 | 達成條件 |
|------|------|------|----------|
| 社交 | `PARTNER_FIRST` | 初結夥伴 | 擁有 1 位夥伴 |
| 社交 | `PARTNER_3` | 黃金同行 | 擁有 3 位夥伴 |
| 社交 | `PARTNER_5` | 戰隊成形 | 擁有 5 位夥伴（達上限） |
| 競爭 | `PARTNER_BEAT_RATE` | 後來居上 | 本月達成率超越任一夥伴 |
| 競爭 | `PARTNER_BEAT_STREAK` | 連續超越 | 破曉連續天數超越任一夥伴 |
| 同步 | `PARTNER_SYNC_7` | 七日同行 | 與任一夥伴連續同日打卡 7 天 |
| 同步 | `PARTNER_SYNC_30` | 同行三十 | 與任一夥伴連續同日打卡 30 天 |
| 鼓勵 | `PARTNER_CHEER_10` | 加油大使 | 累積送出 10 次鼓勵 |
| 鼓勵 | `PARTNER_CHEERED_10` | 人氣戰士 | 累積收到 10 次鼓勵 |

**成就觸發時機**：
- **社交類**：接受邀請後立即計算
- **競爭 + 同步類**：每次打卡提交（或修改）後計算
- **鼓勵類**：每次送出鼓勵後計算（送出方解送出類；接收方靜默解收到類）

所有夥伴成就皆顯示在 Dashboard「成就牆」中。hover/tap 徽章可看詳細條件。

## 規則摘要

| 限制 | 數值 |
|------|------|
| 每人夥伴上限 | **5 位** |
| 拒絕後冷卻時間 | **7 天** |
| 每對每日鼓勵次數 | **1 次** |
| 預設鼓勵訊息選項 | **8 條**（無法自訂） |
| 邀請時間窗 | 無（pending 永久有效，直到對方回應或我取消） |
| 競爭/同步成就比較對象 | 「任一夥伴」（超越/同步任一人即可） |

## 設計考量

**為何雙向確認**：避免單方面強加關係。對方按「接受」才正式建立，符合社交禮節。

**為何上限 5 人**：保持「精選夥伴」的稀缺感。目前社群規模下，5 人上限避免「人人皆夥伴」失去意義。

**為何拒絕有冷卻**：減少騷擾。被拒絕後仍可重邀，但給彼此 7 天冷靜期。

**為何鼓勵限預設訊息**：避免私訊化、減少騷擾風險，同時保持輕量互動感。日後若需自訂訊息可單獨擴充。

**為何沒有 LINE 通知**：避免訊息轟炸。所有狀態都在 `/partners` 頁面或 Dashboard 即時呈現，使用者主動查看。

- 3-4 人小組（共同任務、組內互看）
- 歷史鼓勵查詢頁
- 拒絕陌生邀請的開關
