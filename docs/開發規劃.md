> 最後更新：2026-04-14
> 技術棧：Next.js 16 + Supabase + TypeScript

## 專案背景

使用者參與「星光黃金八套餐」健康挑戰群組，每日需完成 8 項任務並打卡記錄。
系統供 21–50 位成員每日回報、追蹤個人達成率，並提供管理員月結與罰款計算功能。

## 遊戲規則

### 八項每日任務（各 1 分，最高 8 分/天）

| # | 任務 | icon | 說明 |
|---|------|------|------|
| 1 | 早睡早起 | 🌙 | 12 前入睡、早上 7 前起床 |
| 2 | 破曉打拳 | 🥊 | 日出後約 10 分鐘內（有連續加分） |
| 3 | 丹氣跑步 | 🏃 | 15 分鐘，雨天可室內 |
| 4 | 曬太陽 | ☀️ | 有照到才算，陰雨天不計分 |
| 5 | 工作 8 小時 | 💼 | 月累計 160 小時以上 |
| 6 | 不吃肉 | 🥗 | 各素食級別皆可 |
| 7 | 寫觀心書 | 📖 | 一階可寫覺察日記 |
| 8 | 做淨心功法 | 🧘 | 睡前效果更好 |

### 連續破曉打拳加分

- 昨天有打拳 + 今天也打拳 → 今天額外 **+0.5 分**（連續 2 天即觸發）
- 中斷一天後重新計算連續天數
- **每月「破曉王」統計**：當月連續打拳最長天數排名

### 達成率計算

```
有效起始日  = max(月份第一天, 加入日期)
有效滿分    = 有效天數 × 8
達成率      = 累計總分（含加分）÷ 有效滿分 × 100%
```

### 戰士階梯與罰款

| 階梯 | 達成率目標 | 未達罰款 |
|------|-----------|---------|
| 黃金戰士 | 80% 以上 | NT$200 |
| 白銀戰士 | 70% 以上 | NT$300 |
| 青銅戰士 | 60% 以上 | NT$400 |

### 補報規則

- **隔日中午 12 點前**可補報前一天（超過則當日計 0 分）
- 已有記錄者拒絕重複提交

### 每月階梯切換

- 25 日後才開放「選擇下月階梯」入口
- 月結時自動套用 `next_level`，並清空欄位

## API 規格

### 共通規範

- 所有回應格式：`{ ok: true, data: T } | { ok: false, msg: string }`
- 認證：httpOnly Cookie `token`（JWT HS256）
- 時區：Asia/Taipei（UTC+8）

### /api/auth

| 方法 | 路徑 | 說明 | 參數 |
|------|------|------|------|
| POST | /login | 登入 | `{ name, phone_last3 }` |

| GET | /me | 目前用戶 | — |

## 成就系統（49 項）

### 類型 A — 任務連續類（各 8 項任務 × 4 門檻 = 32 項）

每項任務均有 3/7/30/100 天連續成就，代碼格式：`T{n}_STREAK_{days}`

範例：`T2_STREAK_7` = 破曉星火 🥊（破曉打拳連續 7 天）

### 類型 B — 單日特殊類（3 項）

| 代碼 | 名稱 | 條件 |
|------|------|------|
| FIRST_CHECKIN | 萬里起行 🎉 | 第一次打卡 |
| DAILY_PERFECT | 大滿貫 🌟 | 單日 8 項全勤（base_score = 8） |
| DAILY_PERFECT_BONUS | 金色大滿貫 ⭐ | 全勤 + 打拳加分（total = 8.5） |

### 類型 C — 月度類（6 項，月結時觸發）

| 代碼 | 名稱 | 條件 |
|------|------|------|
| MONTH_PASS | 初次通關 🎖️ | 首次月結通過 |
| MONTH_GOLD | 黃金通關 🥇 | 以黃金戰士通過月結 |
| MONTH_PERFECT | 完美月 💯 | 某月達成率 ≥ 100% |
| MONTH_STREAK_3 | 三月連勝 🔥 | 連續 3 個月通過 |
| MONTH_STREAK_6 | 半年英雄 🏆 | 連續 6 個月通過 |
| DAWN_KING | 破曉王 👑 | 當月連續打拳排名第一 |

### 類型 D — 累積里程碑（8 項）

| 代碼 | 名稱 | 條件 |
|------|------|------|
| CHECKIN_30 | 打卡 30 天 📅 | 累計打卡 30 天 |
| CHECKIN_100 | 打卡百日 📅 | 累計打卡 100 天 |
| CHECKIN_365 | 打卡一年 📅 | 累計打卡 365 天 |

## 頁面功能現況

### / — 登入 / 註冊

- 兩 Tab（登入 / 加入）
- 登入：姓名 + 手機末三碼 → JWT Cookie → redirect /checkin
- 加入：姓名、末三碼、加入日期、選擇階梯 → 建立帳號

### /checkin — 每日打卡

- 顯示北部日出時間及建議打拳截止時間（sunrise API，24h 快取）
- 目前連續打拳天數 + 本月達成率進度條
- 8 項任務 toggle（任務 2 動態顯示日出時間提示）
- 若昨日未報且現在 < 中午 12 點，顯示「補報昨日」入口
- 提交後顯示：當日得分（含加分）+ 新解鎖成就通知

### /dashboard — 個人儀表板

- 本月進度卡片（得分 / 目標 / 達成率 / 距通過差幾分）
- 月曆視圖（灰=未報、紅=0-4、橘=5-6、綠=7、金=8+）
- 八項任務本月完成次數長條圖
- 連續打拳卡片（目前 + 本月最長）
- 25 日後顯示「選擇下月階梯」按鈕
- 成就牆（49 格，已解鎖彩色 / 未解鎖灰色）

### /schedule — 行程模板

- **左欄**：標籤庫（系統 14 + 自訂）、可搜尋、可新增自訂標籤（emoji + 色彩）
- **右欄**：時間區段列表（依開始時間排序、跨午夜標示「翌日」）
- **拖拉功能（@dnd-kit）：**
  - 標籤庫 → 時間區段（新增標籤）
  - 時間區段 → 另一區段（移動標籤）
  - 區段內重新排序
  - 拖回左欄（移除標籤）
- 公開 / 私密切換
- 群組行程：瀏覽所有成員公開模板

### /admin — 管理後台

- **進度 Tab**：全員表格（姓名/階梯/得分/達成率/連續打拳/破曉王）
- **罰款 Tab**：未達標成員 + 罰款總計 + 一鍵月結（需確認）
- **會員 Tab**：新增 / 停用成員

### 3. LINE Login 帳號綁定

**需求：** 成員登入後可綁定 LINE 帳號，未來可延伸推播通知、以 LINE 身份快速登入。`members.line_user_id` 欄位已預留。

**OAuth 流程：**
```
1. 成員在儀表板點擊「綁定 LINE 帳號」
2. 前端呼叫 /api/auth/line/authorize → 回傳 LINE OAuth 授權 URL
3. 導向 LINE 授權頁（https://access.line.me/oauth2/v2.1/authorize）
4. LINE 回調至 /api/auth/line/callback?code=...&state=...
5. 用 code 換取 access_token → 呼叫 LINE Profile API 取得 userId / displayName / pictureUrl
6. 將 line_user_id 寫入 members 表，並可選擇性儲存 line_display_name / line_picture_url
7. 導回 /dashboard，顯示綁定成功
```

**新增 DB 欄位（members 表）：**
- `line_display_name TEXT` — LINE 顯示名稱（選填）
- `line_picture_url TEXT` — LINE 頭像 URL（選填）

**新增 API：**
- `GET /api/auth/line/authorize` — 產生並回傳 LINE OAuth URL（附 state 防 CSRF）
- `GET /api/auth/line/callback` — 處理 OAuth callback，寫入 line_user_id，redirect /dashboard
- `DELETE /api/auth/line/unbind` — 解除綁定（清空 line_user_id）

**新增環境變數：**
```
LINE_CHANNEL_ID=
LINE_CHANNEL_SECRET=
LINE_CALLBACK_URL=https://your-domain/api/auth/line/callback
```

**儀表板 UI 新增：**
- 已綁定：顯示 LINE 頭像 + 顯示名稱 + 「解除綁定」按鈕
- 未綁定：「綁定 LINE 帳號」按鈕（LINE 品牌綠色）
