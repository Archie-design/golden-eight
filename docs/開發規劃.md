# 黃金八套餐定課打卡系統 — GAS 網頁應用程式開發規劃

> 建立日期：2026-04-12
> 技術棧：Google Apps Script + Google 試算表

---

## 專案背景

使用者參與「星光黃金八套餐」健康挑戰群組，每日需完成 8 項任務並打卡記錄。
本系統以 Google Apps Script 為後端、Google 試算表為資料庫，
供 21–50 位成員每日回報、追蹤個人達成率，並提供管理員月結與罰款計算功能。

---

## 遊戲規則摘要

### 八項每日任務（各 1 分，最高 8 分/天）

| # | 任務 | 說明 |
|---|------|------|
| 1 | 早睡早起 | 12 前入睡、早上 7 前起床 |
| 2 | 破曉打拳 | 日出後約 10 分鐘內（有連續加分） |
| 3 | 丹氣跑步 15 分鐘 | 雨天可室內跑步 |
| 4 | 曬太陽 | 有照到才算，陰雨天不計分 |
| 5 | 工作 8 小時 | 月累計 160 小時以上 |
| 6 | 不吃肉 | 各素食級別皆可（海鮮素、蛋奶素、五辛素、全素） |
| 7 | 寫觀心書 | 一階可寫覺察日記 |
| 8 | 做淨心功法 | 睡前效果更好（嗯啊哞、當下之舞等） |

### 連續破曉打拳加分

- 昨天有打拳 + 今天也打拳 → 今天額外 **+1 分**（連續 2 天即觸發）
- 中斷一天後重新計算連續天數
- **每月「破曉王」統計**：當月連續打拳最長天數排名

### 達成率計算

- 達成率 = 累計得分（含加分）÷ (入會天數 × 8) × 100%
- 入會天數 = 結算日 − 加入日 − 1（不含今日）
- 加分計入分子，分母不變

### 戰士階梯與罰款

| 階梯 | 達成率目標 | 未達罰款 |
|------|-----------|---------|
| 黃金戰士 | 80% 以上，月約 192-198 分 | 200 元 |
| 白銀戰士 | 70% 以上，月約 168-173 分 | 300 元 |
| 青銅戰士 | 60% 以上，月約 144-148 分 | 400 元 |

### 每月階梯自選（25 日~月底）

- 成員在 25 日~月底可至系統更改「下月挑戰階梯」
- 每月 1 日自動套用新階梯
- 25 日前不開放修改入口

### 重要規則

- **隔日中午 12 點前**可補報前一天（超過則當日計 0 分）
- 每月最後一天進行月結
- 中途退出仍需繳交當月罰款
- 罰款用於群組聚會，多餘捐獻體系基金

## 核心業務邏輯

### 補報規則
```
提交時判斷：
  若 now.hour < 12 且 requestDate == yesterday → 允許補報
  若 requestDate == today → 正常打卡
  其他 → 拒絕，顯示錯誤訊息
已有同日記錄 → 拒絕重複提交（顯示已打卡時間）
```

### 連續打拳加分
```
查詢該會員昨日打卡記錄中任務2是否為 1：
  是 → 今日打拳加分 = 1，連續天數 = 昨日連續天數 + 1
  否 → 今日打拳加分 = 0，連續天數 = (今日任務2 == 1) ? 1 : 0
```

### 達成率計算
```
入會天數 = today - joinDate - 1（不含今日）
有效滿分 = 入會天數 × 8
達成率   = totalScore（含加分）/ 有效滿分 × 100%
```

### 月結流程
```
每月最後一天或管理員手動觸發：
  for each active member:
    1. 計算當月有效天數（入會日起算）
    2. 加總當月總得分（基礎 + 打拳加分）
    3. 計算達成率
    4. 對照挑戰階梯門檻 → 通過/罰款
    5. 計算本月最長連續打拳天數 → 選出破曉王
    6. 寫入 MonthlySummary sheet
    7. 鎖定該月記錄（設標記，不允許再補報）
  觸發月度類成就檢核
```

### 成就檢核
```
function checkAchievements(memberId, date):
  取得該會員所有打卡記錄
  計算各任務當前連續天數（連續類）
  計算當日得分（單日類）
  計算累計打卡次數、大滿貫次數（累積類）
  月結時另行觸發月度類

  for each achievement in ACHIEVEMENT_LIST:
    if 條件成立 AND NOT 已解鎖:
      寫入 Achievements sheet
      回傳新解鎖成就列表（前端顯示 toast 通知）
```

## 技術決策

| 項目 | 選擇 | 原因 |
|------|------|------|
| 前端框架 | Bootstrap 5 CDN | GAS 環境限制，無需打包 |
| 密碼儲存 | SHA-256（Utilities.computeDigest） | GAS 內建，不存明文 |
| Session | PropertiesService (UserProperties) | GAS 標準方式 |
| 部署 | Deploy as Web App（Anyone with link） | 成員透過連結開啟，無需 Google 登入 |
| 日出資料 | 硬編碼於 SunriseData.gs | 離線可用，2026 全年北部資料 |
| 頁面切換 | doGet() 讀取 ?page= 參數 | GAS SPA 標準路由方式 |
| 前端互動 | google.script.run | GAS 前後端標準通訊方式 |
