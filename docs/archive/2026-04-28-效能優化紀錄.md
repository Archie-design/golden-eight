## 5. Dashboard fetch 加 AbortController ✅

### 已完成修改

✅ [app/(main)/dashboard/page.tsx](../app/(main)/dashboard/page.tsx) 重寫 useEffect：

```tsx
useEffect(() => {
  const ac = new AbortController()
  Promise.all([
    fetch('/api/stats/dashboard', { signal: ac.signal }).then(r => r.json()),
    fetch('/api/stats/history',   { signal: ac.signal }).then(r => r.json()),
  ]).then(([dash, hist]) => {
    if (dash.ok) setData(dash); else toast.error(dash.msg)
    if (hist.ok) setHistory(hist.history)
  }).catch(e => {
    if (e.name !== 'AbortError') console.error('[dashboard] fetch failed', e)
  })
  return () => ac.abort()
}, [])
```

額外效益：兩支 API 改為 `Promise.all` 平行發送（原為串連觸發），TTI 略快。

## 10. SW 快取任務圖示 ✅

### 已完成修改

✅ [public/sw.js](../public/sw.js) cache 版本升級為 v2，新增獨立的 `golden-eight-icons-v1` cache：

```js
const CACHE = 'golden-eight-v2'
const ICONS_CACHE = 'golden-eight-icons-v1'

self.addEventListener('fetch', e => {
  if (e.request.method !== 'GET') return
  if (e.request.url.includes('/api/')) return

  // 任務圖示走 cache-first（圖檔不變動，命中即返回）
  if (e.request.url.includes('/icons/tasks/')) {
    e.respondWith(
      caches.match(e.request).then(cached => {
        if (cached) return cached
        return fetch(e.request).then(res => {
          const clone = res.clone()
          caches.open(ICONS_CACHE).then(c => c.put(e.request, clone))
          return res
        })
      })
    )
    return
  }

  // 其他資源：network-first，離線時 fallback 到 cache
  e.respondWith(
    fetch(e.request)
      .then(res => {
        const clone = res.clone()
        caches.open(CACHE).then(c => c.put(e.request, clone))
        return res
      })
      .catch(() => caches.match(e.request))
  )
})
```

`activate` handler 同步更新清理規則，避免誤刪 icons cache。
